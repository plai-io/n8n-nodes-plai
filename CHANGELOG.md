# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-26

### Added

- Initial release of n8n-nodes-plai
- Plai Trigger node for real-time webhook-based lead capture
- Support for Meta (Facebook/Instagram) lead ads
- Support for LinkedIn lead ads
- Client ID and Client Secret authentication
- Dynamic page selection for Meta leads
- Plai action node (operations coming soon)

### Features

- **Real-time lead sync**: Receive leads instantly via webhooks
- **Multi-platform support**: Connect both Meta and LinkedIn advertising accounts
- **Easy setup**: Configure credentials and start receiving leads in minutes
- **Reliable delivery**: Built on n8n's robust webhook infrastructure
